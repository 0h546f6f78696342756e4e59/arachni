Overview

Allows users to skip the crawling phase by extracting paths discovered by a previous scan.

It basically sets the 'restrict_paths' framework option to the sitemap of a previous report.

@author: Tasos "Zapotek" Laskos

def run

    print_status( "Found #{@framework.opts.restrict_paths.size} paths." )
