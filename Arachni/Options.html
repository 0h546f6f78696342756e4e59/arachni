Overview

Options class.

Implements the Singleton pattern and formaly defines all of Arachni’s runtime options.

def to_h
    hash = Hash.new
    self.instance_variables.each {
        |var|
        hash[normalize_name( var )] = self.instance_variable_get( var ) 
    }
    hash
end
